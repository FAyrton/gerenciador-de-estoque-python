# Gerenciador-de-Estoque-python
Gerenciador de Estoque v1.0

Projeto de um sistema de gerenciador de estoque simples feito em PYTHON para portfólio e testar conhecimentos técnicos. 

## Ele tem como objetivo demonstrar e aplicar conhecimento em:
1. Lógica de Programação;
2. Uso de Blocos Condicionais (if/elif/else);
3. Uso de Blocos de Repetição (Loops While);
4. Manipulação de Dicionários, Listas e Strings;
5. Código limpo e Uso de Funções.

## TECNOLOGIAS UTILIZADAS:
- Python 3.14
- Módulos Nativos: system, time

## FUNCIONALIDADES:
- Lógica de adição (cadastro), remoção, substituição e atualização de itens em estoque;
- Lógica de validação de usuário com ID e Senha;
- Lógica de sistema burocrático e funcionalidades através de níveis de permissão;
- Visualização de estoque.

## COMO RODAR O PROJETO?
1. **Clone o repositório (ou baixe os arquivos):**
   ```bash```
```git clone https://github.com/FAyrton/gerenciador-de-estoque-python```
2. **Navegue até a pasta do projeto:**
   ```bash```
   ```cd gerenciador-estoque```
3. **Execute o script principal:**
   ```bash```
   ```python gerenciador-de-estoque.py```
*(Não são necessárias bibliotecas externas para este projeto)*
